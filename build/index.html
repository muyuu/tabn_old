  • htmlにタブを書く

    <div class="tabwrap">
  <ul class="tab">
    <li>タブ1</li>
    <li>タブ2</li>
    <li>タブ3</li>
  </ul>
  <ul class="tabcontent">
    <li>中身ですよ、中身。文句あっか</li>
    <li>中身ですよ、中身。文句あっか</li>
    <li>中身ですよ、中身。文句あっか</li>
  </ul>
</div>

    css はお好みで。

    jQueryを読み込む

    <script src="//code.jquery.com/jquery-1.10.1.min.js"></script>

    tabn.jsを読み込む

    <script src='/tabn.min.js'></script>

    準備は以上なります！

  • タブとタブのコンテンツをラップした要素をセレクターに指定

    $(".tabwrap").tabn();

    アクティブタブとコンテンツには .active が付与される

    タブはデフォルトはセレクタ内の .tab 。変更したい場合はオプションを付与。

    $(".tabwrap").tabn({tab:".tabtab"});

    タブの中身はデフォルトはセレクタ内の .tabcontent 。変更したい場合はオプションを付与。

    $(".tabwrap").tabn({tabcontent:".nakami"});

    アニメーションでの切り替えが可能

    $(".tabwrap").tabn({animation:true});

    cssでアクティブ以外を非表示としている場合は tabn({hide:false})

    $(".tabwrap").tabn({hide:false});
  • tabタブの要素（デフォルトは .tab ）
    contentタブの中身の要素（デフォルトは .tabcontent ）
    hidecssでの削除可否（デフォルトは true ）
    animationアニメーション可否（デフォルトは false ）
    hidetimeアニメーションの消える秒数（デフォルトは 200 ）
    disptimeアニメーションの表示する秒数（デフォルトは 400 ）
