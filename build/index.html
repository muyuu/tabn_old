  • htmlにタブを書く

    <div class="tabwrap">
  <ul class="tasb">
    <li class="tab">タブ1</li>
    <li class="tab">タブ2</li>
    <li class="tab">タブ3</li>
  </ul>
  <ul class="tabcontents">
    <li class="tabcontent">中身ですよ、中身。文句あっか</li>
    <li class="tabcontent">中身ですよ、中身。文句あっか</li>
    <li class="tabcontent">中身ですよ、中身。文句あっか</li>
  </ul>
</div>

    css はお好みで。

    jQueryを読み込む

    <script src="//code.jquery.com/jquery-1.10.1.min.js"></script>

    tabn.jsを読み込む

    <script src='/tabn.min.js'></script>

    準備は以上になります！

  • タブとタブのコンテンツをラップした要素をセレクターに指定

    $(".tabwrap").tabn();

    アクティブタブとコンテンツには .active が付与される

    タブはデフォルトはセレクタ内の .tab 。

    タブをラップする要素はデフォルトはセレクタ内の .tabs 。変更したい場合はオプションを付与。

    $(".tabwrap").tabn({tabs:".tabtab"});

    タブの中身はデフォルトはセレクタ内の .tabcontent。

    タブの中身をラップする要素はデフォルトはセレクタ内の .tabcontents 。変更したい場合はオプションを付与。

    $(".tabwrap").tabn({tabcontents:".nakami"});

    アニメーションでの切り替えが可能

    $(".tabwrap").tabn({animation:true});

    cssでアクティブ以外を非表示としている場合は tabn({hide:false})

    $(".tabwrap").tabn({hide:false});
  • tabsタブの要素（デフォルトは .tabs ）
    contentsタブの中身の要素（デフォルトは .tabcontents ）
    hidecssでの削除可否（デフォルトは true ）
    animationアニメーション可否（デフォルトは false ）
    hidetimeアニメーションの消える秒数（デフォルトは 200 ）
    disptimeアニメーションの表示する秒数（デフォルトは 400 ）
